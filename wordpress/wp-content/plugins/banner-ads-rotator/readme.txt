== Description ==

Easy way to display random advertisement banner on same place. use via short-code anywhere inside content or sidebar.


### Banner Ads Rotator by http://pickplugins.com

* [Live Demo! &raquo;](http://www.pickplugins.com/demo/banner-ads-rotator/)

<br />


<strong>Plugin Features</strong><br />

* Unlimited ads Banner anywhere.
* Use via short-code.
* Trace serve stats and click stats.
* Custom Banner size.
* City and Country stats on click and serve.
* Image, SWF, youtube Video Banner.
* HTML code Banner.




== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Banner Ads Rotator</strong>" activate it.<br />

After activate plugin you will see "Ads Banner" menu at left side on WordPress dashboard click "New Ads Banner" and use the options field "Banner Options"<br />

<br />
<strong>How to use on page or post</strong><br />
When Banner options setup done please publish Ads Banner<br />

and then copy shortcode from top of <strong>Shortcodes Tab</strong> `[bar id="6"]`<br />

then paste this shortcode anywhere in your page to display accordions<br />
